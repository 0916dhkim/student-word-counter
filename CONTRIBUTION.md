# 기여

무엇이든 얼마든지 언제든지 환영입니다. 자소서 쓰는 중간에 살짝 만든거라 코드가 부족한 부분이 많습니다.

## 시스템 요구사항

- Node.JS
- yarn

## 실행 방법

- yarn으로 패키지들을 설치해줍니다.
  ```
  yarn
  ```
- 파이어베이스 관련 환경변수들을 설정해줍니다. 파이어베이스 콘솔에서 앱을 만든 후 Project Settings > Your apps 안의 정보를 `.env.local` 파일에 저장하면 됩니다. dotenv 파일 형식은 `.env.local`을 참조해주세요.
- 개발서버를 실행합니다.
  ```
  yarn start
  ```
